# AI Tools Setup

## What Tools I Installed
- Cursor IDE — an AI-powered code editor
- Claude Code for VS Code — AI coding assistant by Anthropic
- Codex by OpenAI — AI coding agent by OpenAI

## Steps I Completed
1. Downloaded and installed Cursor from cursor.com
2. Installed the Claude Code extension via Cursor's Extensions panel
3. Installed the Codex extension via Cursor's Extensions panel
4. Created a GitHub account and a public repository
5. Installed Git using Xcode Command Line Tools
6. Cloned the repository using Terminal
7. Opened the repo in Cursor and edited this README file

## Issues I Ran Into and How I Solved Them
- Git was not installed on my Mac. I ran xcode-select --install in Terminal to fix it.
- The Git: Clone command was not showing in Cursor, so I used Terminal to clone the repo instead.
- Opening the repo in Cursor required using File > Open Folder manually.
